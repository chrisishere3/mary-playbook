## 🔥 Quick Tips

<Card title="Discovery drives everything">
If you rush it, you lose the deal. Marketing's version of this: if you skip the audience tagging logic, your dashboards lie.
</Card>

<Card title="Activated doors, not MQLs">
Every channel decision should ladder back to cost-per-activated-door. MQL count is a vanity metric. Doors pay the bills.
</Card>

<Card title="Pull live numbers, don't trust the doc">
Numbers in this playbook are reference baselines. Always pull from Metabase or the Q2 MQLs sheet before quoting in a forecast.
</Card>
